== Description ==
Hangit is a single shortcode plugin that provides a popup background
for the featured image in a post.  This is most  useful  for product 
pages such as with WooCommerce when you are selling artwork and your
customer would like to see it hanging on their wall or in front of 
a choice of pre-defined backgrounds.


== Installation ==
Unzip into your plugins directory, install directly from 
wordpress plugins site.
